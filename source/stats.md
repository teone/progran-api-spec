# Group Stats

# IMSI stats [/api/stats/imsi/{imsi}/{counts}]

### Get Stats [GET]

+ Parameters
    + imsi: "001020123456275" (string) - Number of the IMSI to retrieve
    + counts: "10" (string) - Value can be 10, 20 or  "mean"

+ Response 200 (application/json)
      
        [
          {
            "Time"      :   123333 
            "DlBitrate" :   10,
            "UlBitrate" :   20
          },
          {
            "Time"      :   127333
            "DlBitrate" :   8,
            "UlBitrate" :   0
          }
        ]
